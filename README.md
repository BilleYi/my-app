# React资讯网站项目

### 项目结构
```
src
    --components    公共组件
        |--Header           头部组件
        |--Footer           底部组件
    --containers    页面组件
        |--MusicReview      评论页面
        |--EnglishCorner    英语角页面
        |--NewsInfo         资讯页面
        |--Detail           详情页面
    --router        路由封装
    --store         Redux
    --utils         axios封装
    --index.js      入口文件
```
